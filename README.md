# Complex System Simulator

C++20 / Qt5 / SFML project for visualization of complex systems and statistical physics simulations.

## Planned modules

- Ising Monte Carlo
- Percolation
- Kuramoto oscillators
- Quant Monte Carlo market simulator
- Optional SIR epidemic model

## Technologies

- C++20
- Qt5
- SFML
- CMake
- Multithreading
